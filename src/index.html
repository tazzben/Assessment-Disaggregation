Assessment Disaggregation
Required Files

To perform this disaggregation, you must specify a pretest and posttest with some set of matched questions. Standard Scantron, Akindi, Quick Key, ZipGrade, Moodle, Google Form quizzes, Canvas, and some Blackboard formatted files can be specified. For more information about the exam file formats, please see the help menu for the documentation.

Optional Files

The assessment map is optional but recommended. This file's columns are "q" (assessment question number in output), "exam1" (pretest question number), "exam2" (posttest question number), and "options" (number of answer options for the given question). Student list limits the students used for the analysis using a singular column: id.
